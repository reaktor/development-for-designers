What is an accessible text input?

Forms are used to collect input from users. An HTML element used in forms is <input>. You can edit the type attribute of inputs to create text fields, radio buttons, checkboxes, date pickers etc.

The default <input> is a single-line text field. It looks like this:

But to make an accessible text field, we also need a <label>. The label is simply text near the text field. It describes what kind of information the user should write in the text field. It looks like this:

And the HTML code looks like this:

        
            <label for="firstName">First name</label>
            <input id="firstName">

As you can see, the for attribute of the label element is equal to the id attribute of the input element. This is how they are bound together! Thus, screen readers know to read the label out loud when the text field is in focus. Also when you click on the label, it focuses on the text field. Try it! This is very useful especially when using small inputs such as radio buttons and checkboxes.
